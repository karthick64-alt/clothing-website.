# Clothing Store Website

A complete, client-ready clothing store website built with HTML, CSS, and JavaScript.

## Features

### 1. Home Page 1 - Modern Clothing Store Landing
- Hero banner with slider
- Featured categories (Men, Women, Kids, Accessories)
- Trending products slider
- Today's deals section
- Customer testimonials
- Newsletter subscription

### 2. Home Page 2 - Luxury Boutique Landing
- Full-screen hero section
- Curated collections (Ethnic, Western, Sportswear)
- Lookbook style layout
- Designer picks
- Social media feed (Instagram style)

### 3. About Us Page
- Brand story
- Mission & Vision
- Team section
- Store values
- Customer testimonials
- Brand timeline

### 4. Shop / Product Listing Page
- Filterable product grid/list
- Filter by category, size, color, price
- Sorting options (Popularity, Newest, Price)
- Grid/List view toggle
- Pagination

### 5. Product Details Page
- Product image gallery with zoom
- Size & color options
- Add to cart / Buy now
- Stock availability
- Product description
- Fabric details & care instructions
- Customer reviews & ratings
- FAQ section
- Related products

### 6. Blog Listing Page
- Blog posts grid
- Search functionality
- Category filters
- Sidebar with recent posts, categories, popular articles

### 7. Blog Details Page
- Full article page
- Sidebar with recent posts, categories, popular articles
- Author info
- Comments section

### 8. Contact Us Page
- Google Map integration
- Contact form
- Support email & phone
- Store address
- Customer care hours
- Quick links

### 9. Shopping Cart
- Cart item list
- Update quantity
- Apply coupon
- Price summary
- Proceed to checkout

### 10. Checkout Page
- Login/Guest checkout toggle
- Shipping details form
- Billing details form
- Payment options (Card, PayPal, COD)
- Order summary

### 11. User Account Pages
- Login / Register
- My Orders
- Wishlist
- Saved Addresses
- Profile Settings

### 12. Footer Section
- Quick links
- Social media icons
- Newsletter subscription
- Privacy Policy / Terms & Conditions links

## How to Use

1. Open `index.html` in a web browser to view the main homepage
2. Navigate through the site using the navigation menu
3. All pages are fully functional with interactive features
4. Cart and wishlist data is stored in browser localStorage

## Features Implemented

- ✅ Responsive design (mobile-friendly)
- ✅ Interactive navigation
- ✅ Product filtering and sorting
- ✅ Shopping cart functionality
- ✅ Wishlist functionality
- ✅ User authentication (simulated)
- ✅ Form validation
- ✅ Image galleries
- ✅ Sliders and carousels
- ✅ Modal notifications
- ✅ Local storage for cart/wishlist

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Notes

- This is a front-end only implementation
- Cart and wishlist data persists in browser localStorage
- User authentication is simulated (no backend)
- Product images use placeholder URLs (replace with actual images)
- Google Maps embed uses a sample location (update with your address)

## Customization

- Update colors in `css/style.css` (CSS variables)
- Replace placeholder images with actual product images
- Update contact information in `contact.html`
- Modify product data in `js/shop.js` and `js/product.js`
- Customize blog posts in `js/blog.js`
